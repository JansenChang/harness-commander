# 业务需求索引

## 这个文件是做什么的

用于汇总所有产品需求文档，帮助团队快速定位具体业务规则和页面流程。

## 适合写什么

- PRD 文档目录
- 功能名称与对应文档链接
- 当前生效版本
- 需求负责人或背景说明

## 推荐用法

- 把它作为产品知识库入口
- 新需求落地时，先登记到这里再细化文档
- AI 在写业务逻辑前，先从这里找到对应 PRD
